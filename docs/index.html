For Windows OS

For Windows OS

1 - Download and install latest Python release from https://python.org/

2 - Download source code from https://github.com/0x1772/Synaptic/

3 - Install pip packages for Synaptic. You can use "pip install -r requirements.txt"

4 - Run "windows_installation.ps1" with powershell. You can start app with "python main.py -g -v"

For Linux OS

1- Install python with "sudo apt install python3 python3-pip"

2- Clone repo with "git clone https://github.com/0x1772/Synaptic.git". Then go to directory with "cd Synaptic"

3- Install requirements.txt with "pip install -r requirements.tx".

4- Run linux_installation.sh with "sudo sh linux_installation.sh". Then run main.py with "python main.py -v -g"
